Unit 3      A Quick Review of JavaScript-A client side scripting

In unit 3 our topic was JavaScript a client-side scripting where we learned what is JavaScript,
its work, where it goes, its variables and data types, loops, arrays and many more.

JavaScript is a text-based programming language used both on the client-side and server-side
that allows you to make web pages interactive. Where HTML and CSS are languages that give structure
and style to web pages, JavaScript gives web pages interactive elements that engage a user. Inline
JavaScript, Embedded JavaScript & external JavaScript are the three types JavaScript. There are
two basic data types: reference types (usually referred to as objects) & primitive types. Arrays
are one of the most commonly used data structures in programming. JavaScript provides two main
ways to define an array: object literal notation & use the Array () constructor.
