Unit 1       Internet Technology & Web Development

In Unit 1 our topic was Internet Technology & Web Development where we discuss on internet
protocols including a layered architecture, internet definition and history which involves
types of websites,uniform resource locator, hypertext transfer protocol, web browsers
and servers. Moreover, we also discussed on web development which involves client-server model,
different types of roles and skills necessary for web development and types of web development
companies.

The work involved in creating a website for the Internet (World Wide Web) or an intranet
is known as web development (a private network). Web development can range from creating
a basic static website with plain text to creating complicated web apps, electronic companies,
and social networking services. Web engineering, Web design, Web content creation, client
liaison, client-side/server-side scripting, Web server and network security settings, and
e-commerce development are some of the more frequent jobs associated with Web development.
