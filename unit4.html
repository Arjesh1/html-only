Unit 4      Introduction to Server Side Development with PHP

In unit 4 our topic was Introduction to Server-Side Development with PHP where we learned on What
Is Server-Side Development, PHP and Program Control.

Server-side development is much more than web hosting: it involves the use of a programming technology
like PHP or ASP.NET to create scripts that dynamically generate content. All of these technologies
share one thing in common: using programming logic, they generate HTML and possibly CSS and JavaScript
on the server and send it back to the requesting browser. PHP (Hypertext Pre-processor) is known as a
general-purpose scripting language that can be used to develop dynamic and interactive websites. It
was among the first server-side languages that could be embedded into HTML, making it easier to add
functionality to web pages without needing to call external files for data.
